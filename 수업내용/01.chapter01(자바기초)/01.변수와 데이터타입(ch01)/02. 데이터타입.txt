데이터 타입의 종류
1. 기본 자료형(primitive data type) -> 8가지
2. 참조 타입(Reference type)


기본 자료형
정수형(1, 10, 100등 소숫점이 없는 것)
크기가 다르다(담을 수 있는 숫자의 크기가 제한되어 있다.)
byte b; 	<- 1바이트 (8비트)
short s; 	<- 2바이트 (16비트)
int i;	<- 4바이트 (32비트)
long l;	<- 8바이트 (64비트)


byte의 크기는 +-128

int의 크기는 약 +-21억 정도

long타입은 정수 뒤에 l을 붙여줘야 함.
헷갈릴 수 있으니 대문자 L을 권장함.
이를 접미사 라고 함.
ex) 25L



문자 자료형
char c;
2바이트 공간에 문자를 담을 수 있는 데이터 자료형
홑 따옴표를 사용하고, 오직 하나의 글자만 변수에 저장 할 수 있다.



실수 자료형(1.0, 1.123, 0.015634)
float	<- 4바이트
double	<- 8바이트

float타입은 접미사 F를 붙여줘야 함.
ex) float name = 0.5f;




논리 자료형
boolean <- 1바이트.
사실을 관찰하고 결과값이 참, 거짓으로 판별
1비트만으로도 데이터를 표시 할 수 있지만, 컴퓨터에서 데이터 처리의 최소 단위가 1바이트임.
